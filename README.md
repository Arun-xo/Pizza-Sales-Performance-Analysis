Pizza Sales Performance Analysis
📌 Objective

Analyze pizza sales data to evaluate revenue trends, demand patterns, and product performance.

🛠 Tools Used

MS SQL Server

Excel (Pivot Tables, KPI Dashboard)

📊 Key KPIs

Total Revenue: $817,860

Total Orders: 21,350

Average Order Value: $38

Total Pizzas Sold: 49,574

Avg Pizzas per Order: 2.32

📈 Analysis Performed

Daily and hourly sales trend analysis

Category-wise and size-wise contribution

Top 5 best-selling products

Bottom 5 worst-performing products

🔎 Key Insights

Peak demand observed between 12 PM–2 PM and 6 PM–8 PM

Friday generated highest number of orders

Large size pizzas contributed maximum revenue

Classic category had highest total orders
